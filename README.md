# SecureTMS Prototype

A polished full-stack prototype for passwordless transport management using FIDO2/WebAuthn-inspired authentication and a modern operations dashboard.

## Highlights
- Unified home and dashboard experience
- Dedicated booking, shipment, and fleet module pages
- Secure authentication flow with admin and audit views
- Responsive, modern interface for logistics operations

## Setup
1. Install dependencies: `npm install`
2. Ensure MongoDB is running locally
3. Start the app: `npm start`
4. Open: `http://127.0.0.1:4000`

## Main pages
- Home + dashboard: `/index.html`
- Bookings: `/bookings/`
- Shipments: `/shipments/`
- Fleet: `/fleet/`
- Auth pages: `/login.html` and `/register.html`
